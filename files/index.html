    Device Sensor Example

    Welcome to the Device Sensor Example!

    This is a simple demonstration to provide access to devices associated with a basic solution while exercising all available API endpoints.

    Begin by first creating a Product and associate it with the solution. After a product is defined, create and activate one or more devices. Devices can be added under the "Devices" tab of your product configuration.

    Information about each device is displayed in the "Available Devices" section.

    A sample product spec file is included with this example. The spec file can optionally be used for creating a new product and associating new devices. The spec defines a device with these sensors (dataports):

    • state
    • temperature
    • humidity

    This solution example is intended to be minimal with only a few endpoints and rendering requirements.
